# kratin_llc
In this project, I build a App for patience by applying the concepts we have learned till now

website:- https://kratin-llc.vercel.app

Technologies used: React JS, JS, CSS, Bootstrap, Routing

objective :- 1)Improve productivity 2) more efficient interactions between patience and doctor. 3) Geting accurate data. 4) provide solutions.

Features:- 1)Implemented different pages like Home, Profile, Doctors details using React components. 2)Implemented different routes for Home, Profile, Doctors details pages by using React Router components Route, Switch, Link.
